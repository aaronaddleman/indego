# Theme Settings — Changelog

## What's New

Dark mode is here! The app now supports Light, Dark, and System themes.

### Added
- Dark mode with full color adaptation
- Theme selector in Settings (Light / Dark / System)
- System mode follows your OS dark/light preference automatically
- Theme persists across sessions
