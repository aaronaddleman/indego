# Kinetic Sanctuary Redesign — Changelog

## What's New

Complete UI redesign with a fresh green/biophilia design system.

### Phase A: Design Foundation
- New green/emerald color palette (replacing indigo)
- Plus Jakarta Sans font
- Material Symbols icons
- Bottom floating navigation (Today/History/Streaks/Settings)
- Tonal surface layering (no borders)
- Glassmorphism nav with backdrop blur
- 2rem rounded card corners

### Phase B: Today (Dashboard)
- Circular progress ring showing daily completion
- Active streak banner
- Redesigned habit cards with check-to-complete
- Day filter strip for filtering by scheduled day

### Phase C: History
- Momentum Heatmap (contribution grid)
- Time range selector
- Peak streak and completion rate stat cards
- Per-habit breakdown with progress bars

### Phase D: Streaks
- Dedicated streaks page
- Per-habit streak detail with chain visualization
- Motivational "Don't break the chain" banner
- 30-day insights

### Phase E: Settings + Polish
- Redesigned settings with notification toggles
- Final polish and dark mode pass
