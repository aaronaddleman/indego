# Tasks & Lists — User Documentation

## What's New

Indago now supports **Tasks** alongside Habits. Track one-time work items, project steps, and notes — organized into lists with priorities and due dates.

### Tasks
- Create tasks with a title, markdown description, due date, and priority (P1-P4)
- Mark tasks complete with a single tap
- Break down work with unlimited subtask nesting
- Completing a parent task completes all its subtasks

### Lists
- All new tasks go to your **Inbox** by default
- Create custom lists to organize by project or context
- Move tasks between lists
- Delete a list and tasks return to Inbox

### Sorting
- Sort tasks by due date, priority, manual order, or creation date
- Sort preference is saved per list

### Descriptions
- Task descriptions support **Markdown** — add links, lists, headings, code blocks
- View rendered markdown on the task detail page

### Navigation
- New top-level navigation: **Habits** | **Tasks** | **Settings**
- Habit sub-tabs (Today, History, Streaks) are now within the Habits section

## How to Use

1. Tap **Tasks** in the bottom navigation
2. Type a task title in the quick-add bar and press Enter
3. Tap a task to see details, edit description, or add subtasks
4. Use the priority badge to set P1-P4
5. Create lists from the list selector to organize your work

## Known Issues / Limitations

- No drag-and-drop reordering (use move up/down)
- No recurring tasks (use Habits for recurring activities)
- No task sharing (planned for Teams milestone)
- Task descriptions are plain Markdown (no WYSIWYG editor)
